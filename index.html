Buzzword

The party game where every clue has one word in common

How to Play

1
Set Up Teams

Add 2+ teams. Each team takes turns guessing.

2
Read the Clues

A card shows a Buzzword at the top and 10 clues below. Each clue is a phrase with the Buzzword missing. Example: Buzzword is "HOT" → clue "___ dog" → answer "HOT dog".

3
Guess the Phrase

The clue reader reads each clue aloud. Team members try to fill in the blank with the Buzzword. Each correct guess = 1 point.

4
Pass or Play

Can't guess? Pass to move to the next clue. You can pass up to 3 times per round. Passed clues are worth 0 points.

5
Beat the Timer

You have 45 seconds per round. Guess as many as you can before time runs out!
